# Installation

## npm (recommended)

```bash
npm install -g git-unpack
```

## npx (no install)

```bash
npx git-unpack /path/to/repo
```

## From source

```bash
git clone https://github.com/MarkJamesHoward/git-unpack.git
cd git-unpack
npm link
```

## Requirements

- Node.js 14.0.0 or higher
- No additional dependencies required
